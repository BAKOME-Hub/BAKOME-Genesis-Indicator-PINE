## ⚡ Features / Fonctionnalités / Características

| Feature | Description |
|--------|-------------|
| 🟢🔴 **Order Blocks** | Détection automatique des OB haussiers et baissiers avec zones persistantes |
| 🟡🟠 **Fair Value Gaps** | Identification des déséquilibres de liquidité (FVG) |
| 🧹 **Liquidity Sweeps** | Détection des chasses aux stop-loss avant retournement |
| 🚀🔻 **Signaux BUY/SELL** | Entrées avec SL/TP calculés dynamiquement (ATR + Risk/Reward) |
| 🎯 **Labels Contextuels** | "OB Touché", "Sweep détecté", "FVG comblé" en temps réel |
| 📊 **Dashboard** | Delta Volume, Tendance, Session en direct |
| ⏰ **Filtre Session** | London 8h-17h UTC (configurable) |
| 📈 **Confirmation HTF** | Multi-timeframe pour filtrer les faux signaux |
| 🔔 **Alertes** | TradingView alerts pour chaque type de signal |
| 0️⃣ **0 Erreurs 0 Warnings** | Code propre, testé dans l'éditeur Pine |

## ⚙️ Quick Install

1. Ouvrir TradingView → **Pine Editor**
2. Copier le contenu de `bakome_genesis_indicator.pine`
3. Coller dans l'éditeur
4. Cliquer **"Add to chart"**
5. Configurer les paramètres dans les ⚙️ Settings
